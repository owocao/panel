# biu-panel v0.1-stable Release Notes

发布日期：2026-06-17

标签：`v0.1-stable`

提交：`37a9ab1 chore: remove temporary search test file`

## 版本概览

`v0.1-stable` 是 biu-panel 的首个稳定里程碑版本。本版本完成了首页导航、收藏夹、搜索、导入导出、备份恢复、登录安全与基础设置等核心个人面板功能，并对前后端架构做了第一轮保守拆分和稳定性加固。

## 主要功能

### 登录与安全

- 支持首次初始化管理员账号。
- 支持登录、退出、当前用户状态检查。
- 支持浏览器会话与“记住登录”。
- 支持登录失败日志记录。
- 保持 15 分钟内失败 5 次的防暴力破解锁定机制。
- 新增后台清理机制：服务启动时清理一次，之后每 24 小时清理一次。
- `sessions` 只清理已过期记录。
- `login_logs` 默认保留 30 天，不影响 15 分钟防暴力破解窗口。

### 首页导航

- 支持导航分组和导航卡片的新增、编辑、删除。
- 支持导航分组和卡片拖拽排序。
- 支持系统设置内导航管理草稿模式，保存后批量应用。
- 支持公网地址、内网地址和优先打开策略。
- 支持打开地址时自动补全协议，避免无协议地址被识别为站内相对路径。
- 支持内网优先 / 公网优先切换和超时回退。
- 支持导航页单独备份与恢复。
- 修复导航卡片右键菜单图标与实际功能不匹配的问题。
- 修复导航卡片右键菜单位置过高的问题。

### 收藏夹

- 支持收藏夹树形层级展示。
- 支持收藏夹按需加载、展开与折叠。
- 支持书签新增、编辑、删除。
- 支持收藏夹和书签拖拽排序。
- 支持书签跨收藏夹移动。
- 支持批量选择、批量移动、批量删除。
- 优化批量操作复选框居中显示。
- 优化收藏夹与书签右键菜单宽度，减少多余留白。
- 收藏夹内书签右键菜单支持“新标签页打开”和“新窗口打开”。
- 书签右键菜单移除旧的“打开”和“批量选择”入口，统一为更清晰的操作项。
- “移动到文件夹”改为“移动”，“设为首页卡片”改为“首页卡片”。
- 优化移动弹窗文案、尺寸、滚动条、下拉框风格和下拉收起行为。
- 移动目标列表支持展示子收藏夹层级。
- 首页卡片目标分组改为下拉选择当前已有分组。

### 搜索

- 收藏夹搜索改为输入即搜索，不需要点击按钮或按回车。
- 搜索输入加入 250ms 防抖，避免频繁请求导致卡顿。
- 支持输入、删除、清空时实时刷新结果。
- 搜索框为空时恢复当前收藏夹内容。
- 避免慢请求返回后覆盖新搜索词结果。
- 搜索失败时提示错误，不清空用户输入。
- 搜索结果支持展示书签所在完整收藏夹路径。
- 普通收藏夹点选浏览时不显示路径，只在搜索结果中显示路径。

### 元数据抓取

- 支持自动抓取网页标题和图标。
- 支持无 `http://` 或 `https://` 前缀地址的兼容处理。
- 抓取到图标时自动切换为图片图标模式，并填入图片地址。
- 当标题输入框已有内容时，自动抓取不会覆盖现有标题。
- 自动抓取超时时间调整为 3 秒。
- 抓取失败时提供明确反馈。

### 导入导出

- 支持导入浏览器标准书签 HTML。
- 书签导入由正则解析重构为基于 `golang.org/x/net/html` 的 DOM 解析。
- 支持解析常见的 `<DL>`、`<DT>`、`<H3>`、`<A>` 和 `<DD>` 节点结构。
- 支持同一收藏夹层级下重复 URL 检测。
- 导入时跳过重复书签并返回跳过统计。
- 支持导出当前书签为浏览器兼容的 HTML 格式。

### 备份恢复

- 支持完整数据备份下载。
- 支持完整数据恢复。
- 支持导航页数据单独备份与恢复。
- 导航恢复流程包含前端二次确认和基础数据校验。

## 架构与代码整理

### 后端

- 拆分部分 HTTP handler，降低 `server.go` 单文件复杂度。
- 已拆分认证、导航、收藏夹基础接口、设置接口、健康检查等模块。
- 新增独立的书签 HTML 导入解析文件。
- 新增 `CleanupExpiredData(logsRetentionDays int) error` 用于清理过期会话和老旧登录日志。
- 新增后台清理任务，服务启动后执行一次，并每 24 小时执行一次。
- 保持现有登录认证和防暴力破解逻辑不变。

### 前端

- 从 `App.vue` 中抽离部分展示和交互组件。
- 已拆出书签行、收藏夹树节点、移动弹窗、右键菜单、设置相关区域等组件。
- 多处原生 `prompt()` 流程改为统一弹窗交互。
- 优化右键菜单、弹窗、下拉框、滚动条和批量选择等细节样式。

## 修复内容

- 修复收藏夹拖拽排序抖动问题。
- 修复展开收藏夹后频繁拖拽造成的状态异常排查遗留问题。
- 修复书签移动下拉框点击目标后不自动收起的问题。
- 修复书签移动下拉框只显示一级收藏夹的问题。
- 修复书签复制链接在部分环境下不可用的问题。
- 修复自动抓取图标必须输入协议才能生效的问题。
- 修复自动抓取失败反馈不明显的问题。
- 修复右键菜单图标、宽度、位置和留白问题。
- 修复搜索结果不显示完整目录路径的问题。
- 修复普通收藏夹列表不应显示路径的问题。

## 已知限制

- 部分读取接口仍未强制登录校验，若公网暴露需要继续安全加固。
- `App.vue` 仍承担较多顶层状态和业务编排职责，后续需要继续组件化。
- 数据库迁移仍以兼容初始化为主，尚未引入版本化 migration。
- `storage_configs` 与 `backup_records` 表已存在，但当前未作为主流程使用。
- S3 相关能力仍处于基础配置和测试阶段，暂未完成完整远端备份闭环。

- Git 标签已推送：`v0.1-stable`。
