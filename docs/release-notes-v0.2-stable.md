# biu-panel v0.2-stable Release Notes

标签：`v0.2-stable`

## 版本概览

`v0.2-stable` 是后续页面优化和性能优化前的稳定基线版本。本版本重点完成前后端结构收口、运行时测试数据清理和必要稳定性修复。

## 主要内容

- 前端 `App.vue` 拆分完成，核心业务逻辑已迁移到 composable、组件和 utils 中。
- 后端 `httpx/server.go` 拆分完成，metadata、assets、S3、书签导入导出、导航备份等模块已拆出。
- 清理测试数据库和运行时数据，避免个人测试数据、测试书签、测试导航和测试上传文件进入稳定基线。
- 修复前端初始化顺序问题，避免设置页依赖在初始化前被访问导致页面打不开。
- 修复书签导入性能问题，导入流程改为事务化处理，减少逐条查询和逐条提交造成的卡顿。
- 后端测试 `go test ./...` 通过。
- 前端构建 `npm run build` 通过。

## 稳定基线说明

该版本作为后续开发的稳定起点。后续页面优化、交互优化、性能优化和更深层架构调整，应基于 `v0.2-stable` 之后继续推进，避免回退当前已确认的模块边界和稳定行为。
